# Banner 技術需求

## 目標
產生一個響應式網頁橫幅 (Banner) 的 HTML 和 CSS 局部代碼，該代碼未來將手動整合到主要網頁中。

## 詳細需求

1.  **響應式網頁設計 (RWD)**:
    *   橫幅必須能夠適應不同的螢幕尺寸。
2.  **中斷點 (Breakpoints)**:
    *   請使用以下螢幕寬度作為中斷點來調整樣式：`992px`, `768px`, `576px`。
3.  **圖片使用**:
    *   請使用佔位符圖片 (Placeholder Image)。
4.  **內容對齊**:
    *   橫幅中的圖片和文字內容（標題、副標題、按鈕等）需要水平和垂直置中。
